# Support

Strata is not available for production or staging use yet.

## Current Support Scope

Use issues for:

- product questions
- documentation feedback
- telemetry scope suggestions
- privacy and security concerns
- roadmap discussion

## Not Yet Supported

Because application code has not been written, Strata does not yet support:

- installation
- Laravel integration
- dashboard setup
- package upgrades
- production or staging troubleshooting

Those sections will be added before the first usable release.

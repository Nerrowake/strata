# Support

Strata is not ready for production or staging use yet.

## Current Support Scope

Use issues for:

- product questions
- documentation feedback
- telemetry scope suggestions
- privacy and security concerns
- roadmap discussion

## Not Yet Supported

Because Strata is still an early package prototype, it does not yet support:

- installation
- Laravel integration
- dashboard setup
- package upgrades
- production or staging troubleshooting

Those sections will be added before the first usable release.
